# Employee Data Analysis Project

This repository contains a Google Colab notebook for comprehensive analysis of employee information for ABC company. The project involves data preprocessing, exploratory data analysis, and visualization to gain insights into the employee dataset.

## Project Overview

The main goal of this project is to analyze various aspects of employee data, including:
- Employee distribution across teams/departments.
- Predominant age groups.
- Salary expenditure by department.
- Correlation between age and salary.

## Dataset

The analysis is performed on an employee dataset contained in `Employee_Projects.xlsx`. This dataset includes information such as `EmployeeName`, `Gender`, `Age`, `Department`, `Salary`, `YearOfExperince`, and `NumberOfProject`.

## Analysis Tasks Performed

1.  **Employee Distribution Across Teams:** Determined the number and percentage of employees in each department, visualized using a pie chart.
2.  **Segregation by Departments:** Counted employees based on their respective departments.
3.  **Predominant Age Group:** Identified the most frequent age and age group among employees, visualized with a histogram.
4.  **Highest Salary Expenditure:** Calculated total salary expenditure per department and identified the department with the highest expenditure, visualized with a bar chart.
5.  **Age and Salary Correlation:** Investigated the correlation coefficient between 'Age' and 'Salary' and visualized their relationship using a scatter plot.

## How to Run the Notebook

1.  **Upload Data:** Ensure the `Employee_Projects.xlsx` file is uploaded to your Colab environment or the `/content/` directory.
2.  **Open in Colab:** Open the `.ipynb` notebook file in Google Colab.
3.  **Run Cells:** Execute the cells sequentially to perform the data loading, preprocessing, analysis, and visualization steps.
